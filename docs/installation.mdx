# Installation

Install and configure Nebius Skill for your development environment.

## Prerequisites

- **Operating System**: macOS or Linux (Windows requires WSL2)
- **Claude Code**: Latest version ([download](https://claude.com/claude-code))
- **Nebius Account**: Free account at [nebius.cloud](https://nebius.cloud)

## Install Nebius CLI

The Nebius CLI is required for all infrastructure operations.

### macOS / Linux

```bash
# Download and install
curl -sSL https://storage.eu-north1.nebius.cloud/cli/install.sh | bash

# Reload shell to update PATH
exec -l $SHELL

# Verify installation
nebius version
```

**Output should show:**
```
nebius version X.Y.Z
```

### Windows (WSL2)

```bash
# Inside WSL2 terminal
curl -sSL https://storage.eu-north1.nebius.cloud/cli/install.sh | bash
exec -l $SHELL
nebius version
```

## Setup Nebius Skill in Claude Code

### Automatic (Recommended)

Nebius Skill is automatically loaded when you open Claude Code in a Nebius-enabled environment.

### Manual Setup

If not auto-loaded, add to your Claude Code config:

```bash
# Copy skill to Claude Code directory
cp -r ~/path/to/nebius-skill ~/.claude/skills/nebius
```

Restart Claude Code to load the skill.

## Verify Installation

Run the pre-flight check:

```bash
bash ~/.claude/skills/nebius/scripts/check-nebius-cli.sh
```

**Expected output:**
```
✓ nebius CLI installed
✓ nebius profile found
✓ Project ID set
✓ API connectivity verified
✓ Optional tools: docker ✓, kubectl ✓, jq ✓
```

## Troubleshooting Installation

### "nebius: command not found"

```bash
# Check if installation completed
which nebius

# If not found, manually add to PATH
export PATH="$HOME/.nebius/bin:$PATH"

# Add to shell profile (bashrc or zshrc)
echo 'export PATH="$HOME/.nebius/bin:$PATH"' >> ~/.bashrc
```

### "Permission denied" when installing

```bash
# Ensure proper permissions
mkdir -p ~/.nebius
chmod 755 ~/.nebius

# Retry installation
curl -sSL https://storage.eu-north1.nebius.cloud/cli/install.sh | bash
```

### macOS issues

If you see "cannot verify developer" error:

```bash
# Allow the installer
sudo spctl --add-exemption /path/to/installer

# Or bypass Gatekeeper temporarily
sudo spctl --master-disable
# (Re-enable after: sudo spctl --master-enable)
```
