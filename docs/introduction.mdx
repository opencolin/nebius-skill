# Nebius Skill

Deploy and manage infrastructure on **Nebius AI Cloud** using Claude Code.

## What is Nebius Skill?

Nebius Skill teaches Claude Code how to work with Nebius AI Cloud services. It enables you to:

- Deploy serverless AI endpoints with auto-scaling
- Create GPU and CPU virtual machines
- Setup managed Kubernetes clusters
- Configure networking and security
- Manage container registries
- Handle authentication and IAM

Simply describe what you want to deploy in natural language, and Claude Code handles the CLI commands.

## Supported Services

| Service | Purpose | Use Case |
|---------|---------|----------|
| **AI Endpoints** | Serverless ML inference | Deploy LLMs, OpenClaw agents |
| **Compute VMs** | Virtual machines with GPU | Training, development, inference |
| **Managed Kubernetes** | Enterprise k8s clusters | Production workloads, scaling |
| **Container Registry** | Docker image storage | Version control, deployment |
| **VPC Networking** | Networks and subnets | Isolation, security |
| **IAM** | Authentication & access | Service accounts, permissions |

## Key Features

✨ **Natural Language Interface** — Describe what you want, let Claude handle the CLI

🚀 **Quick Deployment** — Endpoints in minutes, clusters in ~10 minutes

💾 **Multiple Inference Options** — Token Factory (Nebius native), OpenRouter, HuggingFace

🔒 **Secure Authentication** — Federation, service accounts, API keys

📊 **Enterprise Ready** — Auto-scaling, load balancing, monitoring integration

## Quick Example

```
User: "Deploy OpenClaw endpoint with GPU"

Claude handles:
1. nebius ai endpoint create --platform gpu-h100-sxm ...
2. Waits for endpoint to be RUNNING
3. Extracts IP address and metrics
4. Returns dashboard URL
```

## Pricing

Nebius AI Cloud uses **pay-as-you-go** pricing:

- **CPU instances**: ~$0.10/hour (4 vCPU)
- **GPU instances**: ~$2.50/hour (H100 80GB)
- **Kubernetes clusters**: ~$0.50/hour + per-node costs

See [Nebius Pricing](https://docs.nebius.com/pricing) for details.

## Architecture

```
Claude Code (Your IDE)
    ↓
/nebius skill (Natural language interface)
    ↓
nebius CLI (Command execution)
    ↓
Nebius API (Infrastructure management)
    ↓
Cloud Resources (Endpoints, VMs, Clusters)
```
